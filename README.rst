Run it
------
Requires `Python <https://www.python.org>`_ >= 3.6, GIT and `Poetry <https://poetry.eustace.io/docs/>`_

Clone this repo and make the venv:
::

    git clone https://github.com/paxet/melat.git
    cd melat
    poetry update

Launch:
::

    poetry run melat
